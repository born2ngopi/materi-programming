# c++

## type data

tipe data pada c++ tidak jauh bereda dengan tipe data pada bahasa pemrograman pada umumnya.

### primitive type

| keyword |      type       |
| :-----: | :-------------: |
|  bool   |     boolean     |
|  char   |    character    |
|   int   |     integer     |
|  float  |    floating     |
| double  | double floating |
|  void   |    valueless    |
